Veda Framework Examples

Interactive demos showcasing the Veda reactive web components framework.

  • Reactive Counter Basic reactivity with state, computed properties, and watchers
  • Reactive Expressions Template expressions and fine-grained DOM updates
  • Unsafe Expressions !{ } Full JavaScript in templates: operators, ternary, methods
  • Loop & If Components Conditional rendering and list iteration
  • Array Index Reactivity Reactive array mutations and index access
  • Virtual List Efficient rendering of 10,000+ items with virtualization
  • Virtual Table Semantic HTML table with virtual scrolling and conditional summary

Running Examples

Build the examples first:

cd examples && pnpm install && pnpm build

Then serve from dist/examples/ or use a local server.
