Virtual Table

A semantic HTML table with virtual scrolling — 5,000 rows rendered efficiently using <tbody items="...">.

Key Features Demonstrated

  • Sticky <thead>: Header stays fixed while scrolling — auto-detected table mode applies position: sticky
  • Semantic <tbody items="...">: Uses native table elements with virtual scrolling via spacer tbody elements
  • Semantic <tfoot condition="...">: Conditional sticky footer that accesses parent component state through Virtual
  • Single <table>: No need for separate header/body tables — everything is one semantic table

Usage

<Virtual items="{this.state.items}" height="400" item-height="42">
  <table>
    <thead>
      <tr><th>Name</th><th>Dept</th><th>Salary</th></tr>
    </thead>
    <tbody items="{this.visibleItems}" as="row" key="id">
      <tr>
        <td>{row.name}</td>
        <td>{row.department}</td>
        <td>{row.salary}</td>
      </tr>
    </tbody>
    <tfoot condition="{this.state.showSummary}">
      <tr><td colspan="3">Total: {this.totalFormatted}</td></tr>
    </tfoot>
  </table>
</Virtual>
